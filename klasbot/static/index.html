● Start and Session ▾
  1. Log in with your teacher PIN. Admins use a PIN too.
  2. From Home or the left rail, open Lesson Planning, Class Management, Help, or Admin (admins only).
  3. Watch the small AI status dot in your teacher card and on Home to confirm the model is ready before you generate.
  4. Use Log out at the top-right when you finish, because the kiosk is shared by other teachers.
✎ Lesson Planning ▾
  1. Open Generate Lesson Plan (DLP, SDLP, or DLL) or Generate Assessment (quiz or exam).
  2. Choose the output, format, grade, subject, quarter, topic, and week. For assessments, also fill the Resources field that appears.
  3. Generate the draft, then review it in Preview or open Edit source to tweak the markdown directly.
  4. Save the draft to My Library before printing, sharing, or creating Teaching Aids from it.
✚ Teaching Aids ▾
  1. Save or open a lesson plan first — Teaching Aids attach to a saved lesson.
  2. Open Teaching Aids and choose the target lesson.
  3. Generate a Worked Example, Guided Practice, Answer Key, Board Notes, or Remediation aid. Add a custom request if you need a specific angle.
  4. Edit, save, copy into the lesson, print, share, or delete the aid as needed.
▤ My Library ▾
  1. Open My Library to see your saved lesson plans, quizzes, exams, and teaching aids grouped by subject.
  2. Use the filter row to search by keyword and narrow by type, subject, grade, format, or sort order.
  3. Open any saved item to restore it to the draft workspace, where you can edit, regenerate, print, share, or delete it.
  4. Your library is private to your login — other teachers cannot see your saved outputs.
◉ Class Management ▾
  1. Open All Classes and create a class with a name, grade, optional section, subject, and school year.
  2. Add students to the roster, create assessments, and enter scores per assessment.
  3. Use the class detail tabs — Dashboard, Add Students, Create Assessments, Daily Attendance, Attendance Performance, Student Performance, and AI Insights.
☑ Daily Attendance ▾
  1. Open a class and choose the Daily Attendance tab.
  2. Pick a date with the date picker. Each student row can be marked Present, Absent, or Late.
  3. Save the day to commit the marks. Saved attendance feeds the Attendance Performance tab and AI Insights.
▲ Performance Views ▾
  • Dashboard — quick summary widgets for the class once scores or attendance exist.
  • Student Performance — each student's scores across every assessment in the class.
  • Attendance Performance — present/absent/late counts and rates per student and across the class.
  • If a view is blank, save assessments, enter scores, or save attendance first — the views read directly from saved data.
★ AI Insights ▾
  1. Open a class and choose the AI Insights tab.
  2. Generate Insights asks KlasBot to summarize the class using saved scores and attendance, so make sure the AI status is green first.
  3. Insights stay loaded while you switch tabs within the same session, so re-opening Insights does not regenerate the answer.
  4. If the result says there is nothing to summarize, add students, save scores, and save attendance, then try again.
▢ Grade Quiz Photo ▾
  1. Create a class assessment first — Grade Quiz Photo transfers scores into it.
  2. Open Grade Quiz Photo, pick the class and assessment, then create the batch with total points, grading style, questions, and answer key.
  3. Upload quiz photos or PDFs, then follow the flow: Detect → Grade → Review → Save.
  4. Open each submission, adjust the score if needed, and mark it Reviewed. Only reviewed submissions transfer.
  5. Preview the transfer, then Save scores to write reviewed scores into the assessment.
⇄ Share & Mobile Companion ▾
  1. Click Share on a saved draft or teaching aid to generate a short-lived QR code and link.
  2. On a phone or tablet on the same trusted Wi-Fi, scan the QR to open the mobile companion view.
  3. The mobile companion has Draft, Library, and Generate tabs only — Class Management, Admin, and Help live on the kiosk.
  4. If the link expires or the device loses Wi-Fi, redo the Share/pair step.
⬢ AI Provider Status ▾
  • Green — provider connected and the model is ready to generate.
  • Amber — provider is reachable but the model is unavailable. Wait for it to load, then retry.
  • Red — provider is not connected. Generation and Ask KlasBot will fail.
  • Gray — status has not been checked yet.
  • Press Check AI in the Ask KlasBot panel to re-run the check at any time.
◆ Admin Tools ▾
  • Teacher Admin — add teachers, assign or reset PINs, grant admin access.
  • Curriculum — upload PDFs with optional subject/version, activate/deactivate, delete, and edit or reset the 10-week pacing in the pacing editor.
  • Plan Formats — edit the visible markdown structure for DLP, SDLP, and DLL outputs. KlasBot adds teacher inputs and curriculum context automatically.
! Troubleshooting ▾
  • Curriculum choices missing → ask an admin to check uploaded and active curriculum.
  • AI unavailable → press Check AI here or wait for the model to come back, then retry.
  • Printing unavailable → use Share to create a short-lived QR or link and print from another device.
  • AI Insights, Dashboard, or Performance widgets blank → save scores and attendance first.
  • Kiosk feels slow → confirm other teachers have logged out and the AI status is green.
